About Smart Voice Based Attendence

The main aim of the project is to record the attendance of the students or employees using voice recognition.Python software would be used for the voice recognition attendance system.

The attendance system based on voice recognition that can be used to record the attendance of students & employees because each person has unique characteristic in voice with frequency.This unique character can be recognized and analyzed to store in internal database.
